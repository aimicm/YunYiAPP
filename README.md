# YunYiAPP
# **医疗助手之移动端APP务**    
项目简介：  
&emsp;&emsp;本项目是个人毕业设计医疗小助手中的【移动端部分】，主要包含【疾病自诊服务】【医疗资源推荐】两大模块。   
&emsp;&emsp;本人安卓小白，这也是为了毕业设计死啃出来的，APP原型来源于https://github.com/gnifeifeiing/Peoplehospital，  
借鉴了其的界面框架，其他已实现的诸如疾病自诊、医院推荐、医生推荐等功能均由个人实现。
